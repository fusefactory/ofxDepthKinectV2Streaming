# ofxDepthKinectV2Streaming

OpenFrameworks addon to receive data from kinect v2 over network;

## Compatibility

Tested with OpenFrameworks 0.10 Mac OS.

## Dependency

- ofxNetwork
- ofxGui
- ofxOpenCV

## Examples

See the examples:

- exampleKinectReceiver
- exampleKinectRecorder
